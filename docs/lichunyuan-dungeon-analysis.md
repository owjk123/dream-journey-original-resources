## 二、核心游戏机制

### 2.1 大富翁式走格玩法

游戏采用经典的大富翁式走格子玩法，玩家通过投掷骰子决定前进步数。

```
┌─────────────────────────────────────┐
│  ┌─┐  ┌─┐  ┌─┐  ┌─┐  ┌─┐  ┌─┐    │
│  │X│→ │?│→ │$│→ │X│→ │?│→ │$│    │
│  └─┘  └─┘  └─┘  └─┘  └─┘  └─┘    │
│              ↑                      │
│         [韦小宝]                     │
│                                     │
│  格子类型:                          │
│  X = 障碍/战斗触发                  │
│  ? = 问号/随机事件                  │
│  $ = 金币/奖励                      │
└─────────────────────────────────────┘
```

### 2.2 如意骰子系统
- **最高上限**: 399个
- **功能**: 投掷骰子前进
- **来源**: 可通过游戏内活动获得（如满汉全席等）

### 2.3 格子事件类型

| 格子类型 | 图标 | 效果 |
|---------|------|------|
| 障碍格 | 黄色交叉剑 | 阻止前进或扣血 |
| 问号格 | 蓝色问号 | 随机事件（道具/buff/陷阱） |
| 金币格 | 金色铜钱 | 获得金币奖励 |
| 宝箱格 | 红色/金色宝箱 | 获得装备或道具 |
| 战斗格 | 怪物图标 | 触发战斗 |
| 传送格 | 紫色法阵 | 进入隐藏区域 |
